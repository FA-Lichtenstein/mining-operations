## Key findings

- **Deliverable:** `open-data-apis-mining.html` — 19-source catalog (12 SA, 7 Africa/global), licensing tiers, API/ingestion patterns, per-theme T1–T6 data package recommendations, PR-07 vs PR-07b boundary table, executive summary table, scorecard, Angular+Firebase sketch.
- **Best source per theme:** T1 → D1 + USGS Africa facilities; T2 → Stats SA P2041 + USGS infrastructure; T3 → P2041 production/sales + ME Bulletin; T4 → USGS MCS (CC0) + Minerals Council; T5 → P2041 long series + QES; T6 → WRC Closure Atlas + Global Tailings + Mine Water Atlas.
- **Most reusable license:** USGS Mineral Commodity Summaries 2025 — **CC0 1.0**; ScienceBase REST for ETL.
- **Highest-value SA entity data:** DMPR Operating Mines (D1) — provincial **HTML tables** + PDF; still no official CSV/API.
- **Cadastre/licensing gap (verified):** No government bulk API for mining rights; SAMRAD requires registration; Project Indwe public map only; ICE DMRE suite is paid. PR-07b must not re-research these sources.
- **Strongest T6 stack:** WRC Mine Closure Risk & Opportunity Atlas (65+ layers) + WRC Mine Water Atlas ZIP downloads + Global Tailings Portal (CC BY-NC-SA; bulk CSV on request).
- **No public operational data:** Block schedules, dispatch logs, mine-level reconciliation — all themes need PR-07b synthetic seeds for credible demos (confirms PR-05 Data feasibility = 3).
- **CGS licensing nuance:** Downloadable 1:1M shapefile free for public use; WMS/OneGeology **personal non-commercial only**; commercial requires written permission.
- **DMPR transition:** DMRE split into DMPR + Department of Electricity and Energy (April 2025); Project Indwe replacing SAMRAD province-by-province.
- **Artifact scorecard:** Ukwazi 5, CV 2, Stack 4, Data 3, Diff 4, Effort M.
