## Key findings

- **Deliverable:** `sa-mining-context.html` — sector snapshot, Stats SA commodity basket table, global positioning, provincial demo anchors, DMRE→DMPR regulatory context, SAMREC/SAMVAL acronym reference, KPI tables mapped to T1–T6, demo narrative cheat sheet, PR-09 boundary table, scorecard, Angular+Firebase sketch.
- **Commodity mix:** Coal (24%) and PGMs (23%) lead the production index; gold (16%) and iron ore (12%) follow. 2025 volumes flat (+0.1% y/y) but PGM/gold sales rose on price — volume vs revenue decoupling useful for T3/T4 narratives.
- **Global anchors:** SA ~88% PGM reserves, ~80% manganese reserves, ~72% chromite reserves; dominant employer commodity PGMs (~40% mining jobs per gov strategy doc).
- **Regulatory:** DMRE split into DMPR (minerals/petroleum) + Department of Electricity and Energy (Proclamation 179, Gazette 51140, 27 Aug 2024); legacy `dmre.gov.za` redirects. MPRDA remains licensing backbone; no public cadastre API (PR-07).
- **SAMREC:** Resource/reserve public reporting code (**2016 ed. still mandatory**; no 2024/2025 code release); CRIRSCO-aligned; JSE-listed mandatory; SANS 10320 for coal; CP/Table 1 vocabulary.
- **SAMVAL:** Mineral asset valuation code (2016 ed.); requires SAMREC-compliant base; adds reasonableness principle; Competent Mineral Asset Valuator (SAMVAL **CV**) ≠ curriculum vitae / PR-03 matrix.
- **KPI map:** Plan attainment, strip ratio, fleet utilisation/cycle time (T1/T2); MCF, survey-to-plan variance, dilution (T3); cost/tonne, AISC, IPCC trade-offs (T4); LOM compliance (T5); closure provision, water/AMD (T6).
- **Demo cheat sheet:** Six-row theme → SA setting → lead KPI → regulatory touchpoint table for interview copy.
- **Boundary respected:** No CV ↔ Ukwazi matrix (PR-03); no open-data catalog duplication (PR-07); no synthetic generators (PR-07b).
- **Artifact scorecard:** Ukwazi 5, CV 3, Stack 4, Data 4, Diff 5, Effort S.
